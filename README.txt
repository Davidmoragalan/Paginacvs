CV 3D Animado — Paquete listo para subir a GitHub Pages

Instrucciones rápidas:
1) Descomprime el ZIP.
2) Reemplaza textos en index.html (nombre, email, ciudad).
3) Si quieres modificar el avatar, edita assets/avatar.svg o reemplázalo por tu imagen (mantén nombre).
4) Crea un repo público en GitHub llamado mi-cv o TU_USUARIO.github.io.
   - Si quieres que se sirva en la raíz (https://TU_USUARIO.github.io/), nombra el repo exactly: TU_USUARIO.github.io
5) Sube los archivos (index.html y la carpeta assets) y activa GitHub Pages en Settings → Pages → Source: main / root.
6) Espera unos segundos y abre la URL que GitHub mostrará.

Contenido del ZIP:
- index.html (página principal)
- assets/avatar.svg (avatar 3D estilizado en vector)
